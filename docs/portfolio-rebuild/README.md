# Portfolio Website Rebuild (Pranav Gorathe)

This folder contains planning and execution docs for rebuilding the personal website from:
- Source reference: `https://dev-pranavgorathe.pantheonsite.io/`

## Goals
- Build a faster, cleaner portfolio site
- Keep core branding/content from current site
- Add a maintainable blog system
- Improve SEO, accessibility, and conversion (contact leads)

## Documents
- `MVP.md` — launch scope and acceptance criteria
- `PRD.md` — product requirements and user stories
- `SYSTEM_DESIGN.md` — architecture and technical decisions

## Proposed Stack
- Next.js (App Router)
- Tailwind CSS
- MDX/Content collections for blog
- Vercel deployment
- Optional CMS later (Sanity/Contentful/Notion API)

## Proposed Routes
- `/` Home
- `/projects`
- `/blog`
- `/blog/[slug]`
- `/contact`

## Launch Checklist (High-Level)
1. Finalize content and section copy
2. Build reusable section components
3. Add blog post model + sample posts
4. Add SEO + analytics + sitemap
5. Test responsiveness + performance
6. Deploy to production
