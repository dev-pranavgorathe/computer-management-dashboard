# CONTENT_PLAN.md — Detailed Website Content Plan

This document gives production-ready content direction for each section/page of the new portfolio website.

---

## 1) Brand Foundation

### 1.1 Positioning
**Primary Positioning:**
Web Developer focused on building practical, fast, conversion-oriented websites and dashboards.

**Secondary Positioning:**
WordPress + React/Next.js implementer for business and social-impact teams.

### 1.2 Voice & Tone
- Clear, practical, confident
- Friendly but professional
- Outcome-driven (show results)
- Avoid buzzwords without evidence

### 1.3 Value Pillars
1. Speed & performance
2. Clean UI/UX
3. Reliable delivery
4. Business impact

## 5) SEO Content Guidelines

- Use one clear H1 per page
- Add keyword variation naturally (not stuffing)
- Keep intros concise and specific
- Use internal links:
  - Home -> Projects -> Blog -> Contact
- Add `schema.org` JSON-LD (Person + Article)

## 7) Review Rubric (before publish)

Score each section out of 5:
1. Clarity
2. Credibility
3. Relevance
4. Conversion strength
5. Consistency

If any section is below 4, revise before launch.
